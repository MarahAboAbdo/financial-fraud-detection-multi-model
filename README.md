# Financial Fraud Detection Using Multi-Model Machine Learning

Project to build an intelligent system for detecting fraudulent financial transactions using three different machine learning models.

---

## Dataset

- Source: Kaggle
- Download Link: https://www.kaggle.com/datasets/amanalisiddiqui/fraud-detection-dataset?resource=download
- Size: 6,362,620 transactions
- Target Variable: isFraud (1 = Fraud, 0 = Legitimate)

---

## Models Used

1. Logistic Regression
2. Random Forest
3. XGBoost

---

## Project Contents

- analysis_model.ipynb - Complete code
- Logistic_Regression.pkl - Model 1
- Random_Forest.pkl - Model 2
- XGBoost.pkl - Model 3
- Report_Arabic.docx - Arabic report
- Report_English.docx - English report

---

pip install pandas numpy matplotlib seaborn scikit-learn xgboost joblib

## How to Run

1. Download the dataset from the link above
2. Open analysis_model.ipynb
3. Run all cells
